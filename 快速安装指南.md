# wo1wan 快速使用指南（Switch 畅玩空间网页游戏）

> **最重要的一句话：不要从"相册"打开 hbmenu！**
> 必须用 **按住 R 键启动一个游戏** 的方式打开 hbmenu，浏览器才能用。
> 这是 Switch 系统的硬限制，和软件本身无关。

---

## 为什么之前一直白屏「无法使用此功能」？

Switch 打开 hbmenu（自制软件菜单）有两种方式，效果完全不同：

| 打开方式 | 叫法 | 能开浏览器吗 |
| --- | --- | --- |
| 点"相册"图标进 hbmenu | applet 模式 | ❌ **不能**（内存小，系统禁止开浏览器） |
| **按住 R 键**再点一个游戏进 hbmenu | 标题接管(title override) | ✅ **能**（完整内存 + 浏览器权限） |

官方教程原文（switch.hacks.guide）：
> 用相册打开 hbmenu 限制很多，包括内存小、**无法启动完整网页浏览器**。建议改用"按住 R 键启动游戏"的方式。

**你之前就是从相册进的，所以浏览器打不开。换成下面的方法就好了。**

---

## 你需要准备

- 一台已破解、装好 Atmosphere 大气层 + 最新固件的 Switch
- 一张内存卡（FAT32 或 exFAT 都行）
- 内存卡里**至少装了一个正常游戏**（卡带或数字版都行，用来"按住 R"接管它）

---

## 第 1 步：下载并放好 wo1wan.nro

1. 下载网页壳本体：
   <https://github.com/Zheng-liu-94/wo1wan/releases/download/latest/wo1wan.nro>
2. 复制到内存卡的 `switch` 文件夹里：
   ```
   /switch/wo1wan.nro
   ```
   （没有 `switch` 文件夹就新建一个。）

> 只需要这一个 `.nro` 文件，**不用装任何 NSP**。

---

## 第 2 步：用「按住 R 键」的方式打开 hbmenu（关键！）

在 Switch 主菜单（桌面）上：

1. **按住 R 键不放**。
2. 保持按住 R，点开任意一个**已安装的游戏**（比如塞尔达、马里奥，卡带或数字版都行）。
3. 继续按住 R 不放，直到出现 hbmenu（自制软件菜单）。
   - 如果游戏会让你先选用户/账号：先点游戏 → 等出现选账号画面 → **这时再按住 R** → 选账号 → 一直按住 R 直到 hbmenu 出现。
4. **检查右上角**：
   - 如果右上角有红字 `Applet Mode` → 说明还是相册模式，浏览器开不了，重来一次。
   - 如果**没有**红字 → 成功进入"标题接管"模式，浏览器可用 ✅

---

## 第 3 步：启动 wo1wan，开玩

1. 在 hbmenu 里找到 **wo1wan**，按 A 打开。
2. 浏览器会自动打开「畅玩空间」主页。
3. 需要登录时，屏幕会显示微信/QQ 二维码 → 手机扫码登录。
4. 操作：
   - 左摇杆 = 鼠标指针，按 **A** = 点击
   - 触摸屏游戏可直接用手指点屏幕
   - 按 **B** 关闭浏览器

---

## 常见问题（先自查）

1. **还是白屏「无法使用此功能」**
   → 右上角有没有红字 `Applet Mode`？有就是没进对模式。重来第 2 步，**一定要按住 R 键点游戏**，不能点相册。
2. **按住 R 也进不了标题接管**
   → 参考第 2 步第 3 条：先点游戏、等选账号画面出来、这时再按住 R 选账号，一直按住直到 hbmenu 出现。个别游戏内存不够，换一个更大的游戏试试。
3. **hbmenu 里找不到 wo1wan**
   → 确认 `wo1wan.nro` 放在内存卡的 `/switch/` 文件夹里、没改错名字。
4. **网页打开但游戏点不动**
   → 左摇杆移动指针 + A 键点击；触屏游戏直接点屏幕。需要键盘输入的游戏在掌机上体验有限，属正常。

## 进阶：做成桌面图标（可选，需要你自己的 prod.keys）

如果你希望像 wiliwili 那样在**桌面直接有个图标**点开（不用每次按 R），需要一个 Application 类型的 NSP 桌面图标。但打包这个合法 NSP 必须用**从你自己主机 dump 出来的 `prod.keys` 密钥**（受版权保护，我无法替你生成或提供）。

如果你愿意自己 dump 密钥，可以这样做：
1. 用 Lockpick_RCM 从你的 Switch dump 出 `prod.keys`。
2. 本地安装 devkitPro + hacbrewpack。
3. 参考 wiliwili 的 `scripts/switch-forwarder/pack.sh`：
   ```bash
   hacbrewpack -k prod.keys --titleid 010ff000ffff0002 --titlename wo1wan --noromfs
   ```
   （exefs/control/logo 内容用 `forwarder/` 目录里已备好的 main、main.npdm、control.nacp、icon 即可。）
4. 得到的 NSP 用 DBI/Tinfoil 安装，桌面就会出现图标。

> 之前 Release 里的 `wo1wan.nsp` 是不含 NCA/CNMT 的裸包，**装不了**（会提示"未找到内容元数据"），已知问题。上面「按住 R」的方法不需要它，推荐直接用那个。
