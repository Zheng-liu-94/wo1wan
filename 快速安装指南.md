# wo1wan 快速安装指南（Switch 畅玩空间网页游戏）

> 一句话：装一个桌面图标（`.nsp`），再从桌面点开它，就能在 Switch 上玩「畅玩空间」的网页游戏。
> **千万不要从 hbmenu 直接开 `.nro`** —— 那样会提示「无法使用此功能」，这是 Switch 系统限制，不是软件坏了。

---

## 你需要准备
- 一台已破解、装好 Atmosphere 大气层 + 最新固件的 Switch
- 一张 FAT32 内存卡（exFAT 也行，但 FAT32 更稳）
- 一台电脑（用来下载文件、传进内存卡）

---

## 第 1 步：下载两个文件
打开浏览器，分别下载这两个文件（来自 GitHub Release `latest`）：

| 文件 | 作用 | 下载地址 |
|------|------|----------|
| `wo1wan.nsp` | 桌面图标启动器（Application 类型） | https://github.com/Zheng-liu-94/wo1wan/releases/download/latest/wo1wan.nsp |
| `wo1wan.nro` | 网页壳本体 | https://github.com/Zheng-liu-94/wo1wan/releases/download/latest/wo1wan.nro |

---

## 第 2 步：把文件放进内存卡
1. 把内存卡用读卡器插到电脑。
2. 把 `wo1wan.nro` 复制到内存卡根目录下的 `switch` 文件夹里，重命名为：
   ```
   /switch/wo1wan.nro
   ```
   （如果这个 `switch` 文件夹不存在，就新建一个。）
3. `wo1wan.nsp` **不用放进内存卡文件夹**——它是要"安装"到机器系统里的（见下一步）。先把它放内存卡随便一个地方（比如根目录）。

---

## 第 3 步：在 Switch 上安装 NSP（三选一）
把内存卡插回 Switch，开机进入 Atmosphere。任选下面一种你常用的安装器：

**方法 A：DBI**
1. 打开 DBI → `Install` → `Install from SD card`。
2. 找到 `wo1wan.nsp`，按 A 安装。

**方法 B：特斯拉菜单 (Tesla / sys-module)**
1. 按 `L + 方向下 + 右摇杆按下` 打开 Tesla overlay。
2. 找到 `File Browser` 或 `NSP 安装` 类功能，定位 `wo1wan.nsp` 安装。

**方法 C：Tinfoil**
1. 打开 Tinfoil → `Browse SD` → 找到 `wo1wan.nsp` → 安装。

> 安装过程和装普通游戏 NSP 一模一样，Atmosphere 会接受这个无任天堂签名的桌面图标。
> 装完后桌面会出现一个 **wo1wan 图标**。

---

## 第 4 步：从桌面图标启动
1. 回到 Switch 主菜单（桌面）。
2. 找到 **wo1wan 图标**，点击打开。
3. 浏览器会自动打开「畅玩空间」主页。

> ⚠️ 再次强调：**必须从桌面图标进**，不要从 hbmenu 里找 `.nro` 手动开。
> 从 hbmenu 开会卡在「无法使用此功能」白屏。

---

## 第 5 步：扫码登录，开玩
1. 网页如果要求登录，屏幕上会显示微信 / QQ 二维码。
2. 拿手机扫这个码，在手机上完成登录。
3. 登录成功后，网页里直接点游戏就能玩。
4. 操作：左摇杆 = 鼠标指针，按 A = 点击；触摸屏游戏直接手指点。
5. 按 **B 键** 关闭浏览器回到桌面。

---

## 常见问题（先自查这 4 条）
1. **提示「无法使用此功能」白屏**
   → 你一定是从 hbmenu 开了 `.nro`。删掉它，从**桌面 wo1wan 图标**进。
2. **桌面没有 wo1wan 图标**
   → NSP 没装成功。重新装一遍 `wo1wan.nsp`（用第 3 步任意方法）。
3. **点图标没反应 / 进不去**
   → 确认内存卡里 `switch/wo1wan.nro` 位置对、没改名；重新拷贝一次两个文件。
4. **网页打开但游戏点不动**
   → 左摇杆移动指针 + A 键点击；触屏游戏直接点屏幕。需要键盘输入的游戏在掌机上体验有限，属正常。

---

- 想自己编译？本地装 devkitPro 后 `bash build.sh`；或推送 GitHub 让 Actions 自动出包。
